## Setting up
### Dependencies
`$ npm install nodemon`

`$ npm install express`


 ### Client

*Development*
` $ npm run serve`

*Production*
 `$ npm run build`

### Server

*Development* 
 `$ nodemon dev` 

*Production*
 `$nodemon start`
